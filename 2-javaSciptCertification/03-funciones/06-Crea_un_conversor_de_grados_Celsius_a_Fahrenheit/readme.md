# Crea un conversor de grados Celsius a Fahrenheit

En este ejercicio, escribirás una función que convierta la temperatura de grados Celsius a Fahrenheit. La fórmula para convertir de grados Celsius a Fahrenheit es

- **Example Code**
  
  ```js
  fahrenheit = celsius * (9/5) + 32
  ```

## Historias de usuario

- Debes crear una función llamada `convertCtoF`.

- `convertCtoF` debe tomar un único argumento numérico, que es la temperatura en grados Celsius.

- `convertCtoF` debe devolver un número.
