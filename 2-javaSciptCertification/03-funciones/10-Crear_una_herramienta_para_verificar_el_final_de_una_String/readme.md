# Crear una herramienta para verificar el final de una String

*En este laboratorio, implementarás una función que compruebe si un String termina con el String de referencia dada.*

## Historias de usuario

1. Debes crear una función llamada `confirmEnding` que tome dos parámetros: el String a verificar y el String de referencia.

2. La función debe devolver `true` si la primera String termina con el segunda String, y `false` en caso contrario.

3. No debe utilizar el método `.endsWith();` en su lugar, utilice uno de los métodos de subStrings de JavaScript para lograrlo.
