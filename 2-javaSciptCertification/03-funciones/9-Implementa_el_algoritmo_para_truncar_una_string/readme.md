# Implementa el algoritmo para truncar una string

*En este laboratorio, practicarás cómo truncar un string a una longitud determinada.*

>[!NOTE]
> **Objetivo**: Cumple con las historias de usuario que se indican a continuación y haz que todas las pruebas pasen para completar el laboratorio.

## Historias de usuario

- Debes tener una función `truncateString` que acepte dos argumentos: el primero, un string; el segundo, un número.

- Si la longitud del string es mayor que el número dado, el string debe truncarse para reducir su longitud de modo que sea igual al número dado, y `...` debe añadirse al final del string truncado.

- Si la longitud del string es igual o menor que el número dado, el string debe devolverse sin cambios.
