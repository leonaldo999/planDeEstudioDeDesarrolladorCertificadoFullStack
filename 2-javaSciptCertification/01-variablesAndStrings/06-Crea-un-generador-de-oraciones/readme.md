# Crea un generador de oraciones

En este laboratorio, crearás dos historias diferentes utilizando una plantilla de oraciones. Utilizarás variables para almacenar las distintas partes de la historia y, a continuación, mostrarás las historias en la consola.

>> *Objetivo*: Cumple con las historias de usuario que se indican a continuación y haz que todas las pruebas pasen para completar el laboratorio.

## Historias de usuario

1. Debes declarar las siguientes variables utilizando `let`:

    - `adjective`
    - `noun`
    - `verb`
    - `place`
    - `adjective2`
    - `noun2`

2. Debes asignar a las variables anteriores algunos valores de cadena de tu elección.

3. Debes declarar una variable `firstStory`.

4. Debes utilizar la siguiente plantilla de historia para crear la primera historia y asignarla a la variable `firstStory`: `"Once upon a time, there was a(n) [adjective] [noun] who loved to eat [noun2]. The [noun] lived in a [place] and had [adjective2] nostrils that blew fire when it was [verb]."`;

5. Debes mostrar tu primera historia en la consola utilizando el mensaje `"First story: [firstStory]"`.

6. Debes asignar nuevos valores a tus variables `adjective`, `noun`, `verb`, `place`, `adjective2`, y `noun2`.

7. Debes declarar una variable `secondStory`.

8. Crea otra historia utilizando la misma plantilla y asígnala a la variable `secondStory`.

9. Debes mostrar tu segunda historia en la consola utilizando el mensaje `"Second story: [secondStory]"`.
