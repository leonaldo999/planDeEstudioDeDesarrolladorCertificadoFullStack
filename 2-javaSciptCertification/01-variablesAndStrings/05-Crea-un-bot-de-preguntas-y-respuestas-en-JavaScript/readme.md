# Crea un bot de preguntas y respuestas en JavaScript

>> *Objetivo*: Cumple con las historias de usuario que se indican a continuación y haz que todas las pruebas pasen para completar el laboratorio.

## Historias de usuario

1. Debes registrar `"Hello! I'm your coding fun fact guide!"` en la consola como mensaje de bienvenida para el usuario.

2. Debes crear tres variables: `botName`, `botLocation` y `favoriteLanguage`, que almacenen el nombre del bot, su procedencia y su lenguaje de programación favorito, respectivamente.

3. Debes registrar `"My name is (botName) and I live on (botLocation)."` en la consola.

4. Debes registrar `"My favorite programming language is (favoriteLanguage)."` en la consola.

5. Debes usar `let` para crear una variable `codingFact` y asignarle una cadena que sea un dato curioso sobre el lenguaje de programación favorito de tu bot, e incluir el uso de la variable `favoriteLanguage`.

6. Debes registrar `codingFact` en la consola.

7. Debes reasignar la variable `codingFact` a un nuevo dato sobre el lenguaje favorito del bot usando nuevamente la variable `favoriteLanguage`.

8. Debes registrar `codingFact` en la consola nuevamente.

9. Debes reasignar la variable `codingFact` nuevamente a otro dato nuevo sobre el lenguaje favorito del bot utilizando la variable `favoriteLanguage`.

10. Debes registrar `codingFact` en la consola por tercera vez.

11. Debes registrar `"It was fun sharing these facts with you. Goodbye! - (botName) from (botLocation)."` en la consola como un mensaje de despedida del bot.
