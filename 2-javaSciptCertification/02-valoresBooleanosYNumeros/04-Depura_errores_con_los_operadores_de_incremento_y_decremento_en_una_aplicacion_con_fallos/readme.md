# Depura errores con los operadores de incremento y decremento en una aplicación con fallos

*Un amigo te ha pedido que le ayudes a depurar su código. Está practicando con los operadores de incremento y decremento, pero obtiene resultados inesperados en la consola.*

Tu tarea consiste en ayudarle a identificar los errores y actualizar el código para que los resultados de la consola coincidan con los esperados.

Puedes consultar las lecciones anteriores sobre los operadores de incremento y decremento, así como sobre la notación prefija y posfija.

---

## Historias de usuario

1. Debes tener una variable llamada `updatedCounter` que genere correctamente el valor `11` utilizando el operador de incremento. Actualiza la expresión para utilizar la notación prefija o posfija correcta en lugar de codificar el valor de forma rígida.

2. Debes tener una variable llamada `finalScore` que genere correctamente el valor `8` utilizando el operador de incremento. Actualiza la expresión para utilizar la notación prefija o posfija correcta en lugar de codificar el valor de forma rígida.

3. Debes tener una variable llamada `updatedCoins` que genere correctamente el valor `2` utilizando el operador de decremento. Actualiza la expresión para utilizar la notación prefija o posfija correcta en lugar de codificar el valor de forma rígida.

4. Debes tener una variable llamada `newHealth` que genere correctamente el valor `7` utilizando el operador de decremento. Actualiza la expresión para utilizar la notación prefija o posfija correcta en lugar de codificar el valor de forma rígida.
