# Depura errores de conversión de tipos en una aplicación con fallos

*Acabas de incorporarte a una empresa local de desarrollo web y tu primera tarea consiste en limpiar el código defectuoso que dejaron los desarrolladores anteriores.*

*Intentaban realizar operaciones aritméticas sencillas, pero algo salió mal y los resultados no tienen sentido. Tu tarea es revisar, depurar y corregir el código para que funcione como se espera y sea más fácil de leer.*

**Objetivo**: Cumple con las historias de usuario que se indican a continuación y haz que todas las pruebas pasen para completar el laboratorio.

---

## Historias de usuario

1. Debes tener una variable llamada `firstResult` que sume correctamente los números `5` y `10` para producir el valor `15`.

2. Debes tener una variable llamada `secondResult` que reste correctamente `5` de `8` para producir el valor `3`.

3. Debes tener una variable llamada `thirdResult` que produzca el valor `6` al sumar dos números. Reemplaza el `booleano` que se usa actualmente en la expresión por un número.

4. Debes tener una variable llamada `fourthResult` que produzca el valor `8` al sumar dos números. Reemplaza el `booleano` que se usa actualmente en la expresión por un número.

5. Debes tener una variable llamada `fifthResult` que multiplique correctamente dos números para obtener el producto `20`. Reemplaza el `string` que se usa actualmente en la expresión por un número.

6. Debes tener una variable llamada `sixthResult` que sume correctamente dos números para producir el valor `22`. Reemplaza el valor `null` que se usa actualmente en la expresión por un número.
