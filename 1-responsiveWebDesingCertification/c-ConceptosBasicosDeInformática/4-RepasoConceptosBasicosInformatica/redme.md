## Comprensión de los conceptos básicos de informática, Internet y herramientas para desarrolladores

- **Placa base** : contiene toda la memoria, los conectores y los discos duros necesarios para el funcionamiento del ordenador. Actúa como la placa de circuito principal del ordenador.

- **Unidad central de procesamiento (CPU)** : procesador que se encarga de ejecutar instrucciones y realizar cálculos.

- **Memoria de acceso aleatorio (RAM)** : una ubicación de almacenamiento temporal para la CPU de la computadora.

- **Unidad de disco duro (HDD)** : una ubicación de almacenamiento permanente que se utiliza para almacenar datos incluso cuando la computadora está apagada.

- **Unidad de estado sólido (SSD)** : memoria flash no volátil y se puede utilizar en lugar de un disco duro.

- **Unidad de fuente de alimentación (PSU)** : responsable de convertir la electricidad del tomacorriente de pared en un formato que la computadora pueda utilizar.

- **Unidad de procesamiento gráfico (GPU)** : responsable de representar imágenes en la pantalla de la computadora.

- **Diferentes tipos de proveedores de servicios de internet** : Un proveedor de servicios de internet (ISP) es una empresa que proporciona acceso a internet. Existen diferentes tipos de ISP, como acceso telefónico, DSL, cable, fibra óptica y satélite.

- **Formas seguras de iniciar sesión en su computadora** : algunos ejemplos de formas seguras de iniciar sesión en su computadora incluyen usar una contraseña segura, habilitar la autenticación de dos factores y usar un administrador de contraseñas.
Entorno de desarrollo integrado (IDE) : una herramienta que ayuda a los desarrolladores a escribir, probar y depurar código de manera eficiente.

- **Editor de código** : una herramienta que los desarrolladores utilizan para escribir y depurar código.

- **Git** : un popular sistema de control de versiones que permite a los desarrolladores realizar un seguimiento de los cambios en su código y colaborar con otros.

- **Servicios de alojamiento en la nube para repositorios** : Un repositorio es un lugar de almacenamiento para archivos de proyecto e historial de versiones. Entre los servicios de alojamiento en la nube más populares para repositorios se incluyen GitHub, GitLab y Bitbucket.

- **Gestores de paquetes** : herramientas que ayudan a los desarrolladores a simplificar el proceso de añadir, actualizar y eliminar bibliotecas y dependencias del proyecto. Algunos ejemplos son npm, pip y Maven.

- **Bibliotecas y frameworks de prueba** : Las pruebas se realizan en el software para garantizar que el código funcione según lo previsto. Algunos ejemplos de bibliotecas y frameworks de prueba son Jest, PHPUnit y JUnit.

## Trabajar con archivos, sistemas de archivos y formatos multimedia

- **Prácticas recomendadas para nombrar archivos** : Debe nombrar sus archivos de forma que sean fáciles de entender y mantener. Por ejemplo, `about-us.html` es un nombre más descriptivo que `page1.html`.

- **Directorio raíz** : directorio de nivel superior en un sistema de archivos. Directorio es otro nombre para una carpeta.

- **Markdown** : lenguaje de marcado comúnmente usado para documentación y archivos `README`. Un archivo `README` contiene información sobre un proyecto, como su instalación y uso.

- `index.html`: representa la página predeterminada que se muestra cuando un usuario visita un sitio web.

- **Crea, mueve y elimina archivos y carpetas con el Explorador/Finder** : El Explorador es el administrador de archivos de Windows y el Finder lo es de macOS. Puedes usar estas herramientas para crear, mover y eliminar archivos y carpetas.

- **Búsqueda de archivos y carpetas** : puede utilizar la función de búsqueda en el Explorador o el Finder para encontrar archivos y carpetas en su computadora.

- **Tipos de archivos HTML, CSS y JS** : `.html` la extensión de archivo se utiliza para archivos HTML, `.css` archivos CSS y `.js` archivos JavaScript.

- **Formatos de imágenes y gráficos comunes** : `JPEG` y `PNG` son formatos de archivos de imagen comunes. `GIF` es otro formato de archivo de imagen común que admite animación. `SVG` es un formato de archivo para gráficos vectoriales.

- **Formatos comunes de audio y video** : Este `MP3` formato se usa comúnmente para archivos de audio `MP4` y video . `MOV` Fue desarrollado por Apple y se usa comúnmente para archivos de video.

- **Formatos de fuente comunes** : Este `TTF` formato se usa comúnmente para fuentes TrueType y `WOFF` para fuentes web. El sucesor de WOFF es `WOFF2`, que ofrece una mejor compresión.

- **ZIP** : formato de archivo que se utiliza para comprimir archivos y carpetas.

## Navegar eficazmente por la web

- **¿Qué es un navegador web?** : una aplicación de software que permite a los usuarios acceder y ver sitios web en Internet.

- **¿Qué es un motor de búsqueda?** : Es una herramienta que permite a los usuarios buscar información en internet. Algunos ejemplos son Google, Bing y Yahoo.

- **Navegadores web comunes** : algunos ejemplos de navegadores web comunes incluyen Google Chrome, Mozilla Firefox y Microsoft Edge.

- **Estrategias de búsqueda comunes** : Puede usar `site:` seguido de la URL de un sitio web para buscar contenido en él. Puede usar " `filetype:`" seguido de una extensión de archivo para buscar archivos de ese tipo. Puede anteponer un signo menos a un término de búsqueda para excluir resultados que lo contengan. Puede anteponer un signo más a un término de búsqueda para incluir resultados que lo contengan.
