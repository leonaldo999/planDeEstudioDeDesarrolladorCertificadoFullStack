# Navegar eficazmente por la web

En estos videos de conferencias, aprenderá sobre qué son los sitios web, los motores de búsqueda y los navegadores web, los diferentes navegadores disponibles y cómo sacar lo mejor de un motor de búsqueda.

---

## ¿Cuáles son algunos de los navegadores web más comunes disponibles hoy en día y cómo instalar uno?

Los navegadores web son aplicaciones que se usan para visitar páginas en internet. Si estás cursando el programa de freeCodeCamp, es muy probable que ya estés usando un navegador web.

A partir de 2024, los navegadores más populares fueron `Microsoft Edge`, `Firefox`, `Google Chrome` y `Safari`. Existen muchas otras opciones, y las cuotas de mercado cambian constantemente, pero estos cuatro son los más comunes actualmente.

Lo más probable es que tu sistema operativo venga con un navegador instalado por defecto. En `Windows`, `Microsoft Edge`. En `Mac`, `Safari`. En `Linux`, probablemente `Firefox`.

¿Pero qué pasa si quieres utilizar un navegador diferente?

La buena noticia es que ya tienes un navegador predeterminado. Así que puedes navegar a la página web del navegador que quieras y descargar el instalador adecuado para tu sistema operativo.

Al elegir un navegador, tenga en cuenta factores como la velocidad, las características de seguridad y la compatibilidad con los sitios web que visita con frecuencia.

Algunos navegadores ofrecen funciones únicas. Por ejemplo, Opera tiene una VPN integrada, mientras que Brave se centra en la privacidad y el bloqueo de anuncios.

La instalación suele ser sencilla. Tras descargar el instalador, simplemente ejecútelo y siga las instrucciones en pantalla.

La mayoría de los navegadores le ofrecerán importar sus marcadores y configuraciones desde su navegador predeterminado actual, lo que hará la transición más fácil.

Alternativamente, puede instalar un navegador a través del administrador de paquetes de su sistema operativo, como `brew install --cask google-chrome` en macOS con Homebrew o `yay -S brave-bin` en Arch Linux.

El administrador de paquetes y los nombres de paquetes disponibles variarán según su sistema operativo.

Recuerda que no estás limitado a un solo navegador. Muchas personas usan distintos navegadores para distintos fines: quizás uno para el trabajo y otro para uso personal.

Siéntete libre de experimentar con diferentes navegadores para encontrar lo que funciona mejor para ti.

## ¿Cuál es la diferencia entre un navegador web, un sitio web y un motor de búsqueda?

Los navegadores web son aplicaciones de software que permiten a los usuarios acceder y navegar por la World Wide Web. Interpretan y muestran contenido web, como texto, imágenes, vídeos y elementos interactivos.

Las características principales de los navegadores web incluyen la barra de direcciones, donde se ingresan URL o consultas de búsqueda, y el motor de renderizado, que traduce HTML, CSS y JavaScript en páginas web visuales.

Los navegadores también ofrecen marcadores para guardar y organizar sitios web favoritos, extensiones para mejorar la funcionalidad y varias funciones de privacidad y seguridad, como bloqueadores de ventanas emergentes y modos de navegación privada.

Los navegadores web más populares incluyen Google Chrome, Mozilla Firefox, Microsoft Edge, Safari y Opera.

Los sitios web, por otro lado, son conjuntos de páginas web y contenido relacionado, identificados por un nombre de dominio común y publicados en al menos un servidor web. Son los destinos que visitas con tu navegador web.

Los componentes de los sitios web incluyen el nombre de dominio, que es la dirección única del sitio web (como freecodecamp.org), y las páginas web, que son documentos individuales que forman un sitio web.

Los sitios web también suelen incluir contenido multimedia, como imágenes, vídeos y archivos de audio, que mejoran la experiencia del usuario, así como elementos interactivos, como formularios y botones, que permiten la interacción del usuario.

Los sitios web pueden ser estáticos, donde el contenido rara vez cambia, o dinámicos, donde el contenido se actualiza con frecuencia o se personaliza para los usuarios.

Los motores de búsqueda son herramientas web diseñadas para ayudar a los usuarios a encontrar información en internet. Utilizan algoritmos complejos para indexar y clasificar páginas web según su relevancia para las consultas de los usuarios.

Los aspectos clave de los motores de búsqueda incluyen rastreadores web, también conocidos como arañas o bots, que son programas que navegan sistemáticamente por la web para descubrir e indexar contenido nuevo.

El proceso de indexación implica organizar y almacenar información sobre páginas web en bases de datos masivas.

Los motores de búsqueda también emplean algoritmos de clasificación, que son sistemas complejos que determinan el orden de los resultados de búsqueda según factores como la relevancia, la autoridad y la experiencia del usuario. Entre los motores de búsqueda más populares se encuentran Google, Bing, Yahoo y DuckDuckGo.

La relación entre navegadores, sitios web y motores de búsqueda está interconectada.

Los navegadores web son las herramientas que se utilizan para acceder tanto a sitios web como a motores de búsqueda.

Los sitios web son los destinos que los usuarios visitan a través de los navegadores o encuentran mediante motores de búsqueda.

Los motores de búsqueda ayudan a los usuarios a descubrir sitios web al proporcionar acceso organizado a la gran cantidad de información disponible en la web.

Comprender estas distinciones y relaciones es fundamental para un uso eficaz de Internet y una alfabetización web básica

## Cómo usar un motor de búsqueda eficazmente para lograr resultados óptimos

Buscar información en internet puede ser una tarea abrumadora. Hay tantos sitios web con tanta información que puede parecer imposible obtener los resultados deseados.

Pero gracias a cosas como la sintaxis de búsqueda booleana, puedes limitar tu búsqueda a exactamente lo que necesitas.

La sintaxis de búsqueda booleana es una forma de utilizar palabras especiales para ayudarle a encontrar exactamente lo que está buscando cuando busca en línea.

A continuación se muestra un ejemplo de una consulta de búsqueda básica para `freecodecamp curriculum`.

Cuando buscas `freecodecamp curriculum`, el motor de búsqueda devolverá resultados que coincidan con cualquiera de estas palabras clave, generalmente ordenados por relevancia (la probabilidad de que coincidan o estén relacionados con más aspectos de tu consulta).

¿Pero qué pasa si solo quieres resultados que coincidan con ambas palabras clave? Quizás hayas oído que puedes escribir una consulta entre comillas, como `"freecodecamp curriculum"`.

Sin embargo, esta consulta coincide específicamente con la frase `freecodecamp curriculum` . Esto significa que los resultados que verá se limitarán a los sitios que dicen `freecodecamp curriculum` juntos, no a los que dicen `freecodecamp` Y `curriculum`.

Si desea hacer coincidir sitios que incluyan ambos términos, pero no necesariamente uno al lado del otro como una frase, puede anteponer a cada término un símbolo más, como se muestra a continuación:`+freecodecamp` `+curriculum`

Alternativamente, quizás te interesen las iniciativas de freeCodeCamp fuera del currículo. En ese caso, puedes anteponer el término `curriculum` con un símbolo menos, como este: `+freecodecamp -curriculum`.

Esto devolverá resultados que mencionen `freecodecamp`, pero excluirá resultados que también mencionen `curriculum`.

¿O quizás buscas en un sitio específico, como nuestra plataforma de noticias? Puedes usar `site:` seguido de la URL del sitio: `site:freecodecamp.org/news curriculum`.

Esta consulta devuelve todos los resultados `freecodecamp.org/news` que contienen la palabra `curriculum`.

Al combinar estas herramientas, puedes realizar consultas muy específicas para obtener la información exacta que necesitas.

Por ejemplo, la consulta `site:freecodecamp.org/news +python -curriculum` buscaría artículos de Python en nuestro sitio de noticias, excluyendo artículos que pudieran hablar sobre nuestro plan de estudios.

Los operadores de búsqueda pueden variar, por lo que siempre es importante consultar la documentación de su motor de búsqueda preferido.

Pero ahora deberías tener la capacidad de obtener los resultados exactos que deseas de tus búsquedas web.
