# Crea una página de recopilación de vídeos

---

**Objetivo**: Cumplir las historias de usuario a continuación y lograr aprobar todas las pruebas para completar el laboratorio.

**Historias de usuarios**:

- Debe tener un elemento `main` como único hijo del elemento `body`.

- Debes tener un elemento `h1` con el tema de tu página.

- Debes tener un párrafo que presente el tema de tu página debajo de tu elemento `h1`.

- Debes tener tres elementos `section` debajo de tu primer párrafo.

- Cada sección debe contener un elemento `h2`, un párrafo y un elemento `iframe`, en este orden.

- Los tres elementos `iframe` deben tener un atributo `src` establecido en un vídeo válido.

- Cada elemento `iframe` también debe tener un atributo `title` para describir el contenido incrustado y un atributo `height` y un atributo `width` para establecer el elemento en un tamaño adecuado.
