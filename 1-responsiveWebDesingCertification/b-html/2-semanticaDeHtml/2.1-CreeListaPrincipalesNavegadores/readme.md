# Cree una lista de los principales navegadores web

En este desafío, creará una lista no ordenada de los principales navegadores web utilizando HTML.

---

## Paso 1

En este taller, vas a crear una lista de los principales navegadores web. Se te ha proporcionado el código HTML básico.

> Comienza añadiendo un encabezado a tu página con el texto `List of Major Web Browsers` utilizando un elemento `h1` dentro del elemento `body`.

- **Codigo**

  ```html
  <h1>List of Major Web Browsers</h1>
  ```

---

## Paso 2

Como recordarás de una lección anterior, las listas descriptivas se utilizan para presentar términos y definiciones en un formato organizado y fácil de leer.

Aquí tienes un ejemplo:

- **Codigo Ejemplo**

  ```html
  <dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
  </dl>
  ```

> Debajo del elemento `h1`, crea un elemento `dl`. Este contendrá la lista de navegadores.

- **Codigo**

  ```html
  <h1>List of Major Web Browsers</h1>
  <dl></dl>
  ```

---

## Paso 3

Las listas descriptivas están compuestas por una lista de términos y detalles. El elemento `dt` representa el término descriptivo, mientras que el elemento `dd` representa el detalle descriptivo.

El primer navegador que agregará a su lista descriptiva será `Google Chrome`.

> Dentro de su elemento `dl`, agregue un elemento `dt` que contenga el texto `Google Chrome`.

Debajo de su elemento `dt`, agregue un elemento `dd` con el siguiente texto: `This is a free web browser developed by Google and first released in 2008.`

- **Codigo**

  ```html
  <h1>List of Major Web Browsers</h1>
  <dl>
    <dt>Google Chrome</dt>
    <dd>This is a free web browser developed by Google and first released in 2008.</dd>
  </dl>
  ```

---

## Paso 4

Ahora es el momento de añadir tu segundo navegador a la lista.

Debajo de la entrada de `Google Chrome`, añade otro elemento dt que contenga el texto `Firefox`.

Debajo del elemento `dt`, añade un elemento `dd` con el siguiente texto: `This is a free web browser developed by the Mozilla Corporation and first created in 2004.`

- **Codigo**

  ```html
  <h1>List of Major Web Browsers</h1>
  <dl>
    <dt>Google Chrome</dt>
    <dd>This is a free web browser developed by Google and first released in 2008.</dd>
    <dt>Firefox</dt>
    <dd>This is a free web browser developed by the Mozilla Corporation and first created in 2004.</dd>
  </dl>
  ```

---

## Paso 5

El tercer navegador que agregará a la lista será el navegador web `Safari`.

Agregue otro elemento `dt` que contenga el texto `Safari`.

Debajo del elemento `dt`, agregue un elemento `dd` con el siguiente texto: `This browser was developed by Apple and is the default browser for iPhone, iPad and Mac devices.`

- **Codigo**

  ```html
  <h1>List of Major Web Browsers</h1>
  <dl>
    <dt>Google Chrome</dt>
    <dd>This is a free web browser developed by Google and first released in 2008.</dd>
    <dt>Firefox</dt>
    <dd>This is a free web browser developed by the Mozilla Corporation and first created in 2004.</dd>
    <dt>Safari</dt>
    <dd>This browser was developed by Apple and is the default browser for iPhone, iPad and Mac devices.</dd>
  </dl>
  ```

---

## Paso 6

Ahora es el momento de añadir el cuarto navegador a la lista.

Añade otro elemento `dt` que contenga el texto `Brave`.

Debajo del elemento `dt`, añade un elemento `dd` con el siguiente texto: `This is a free web browser first released in 2016 that is based on the Chromium web browser.`

- **Codigo**

  ```html
  <h1>List of Major Web Browsers</h1>
  <dl>
    <dt>Google Chrome</dt>
    <dd>This is a free web browser developed by Google and first released in 2008.</dd>
    <dt>Firefox</dt>
    <dd>This is a free web browser developed by the Mozilla Corporation and first created in 2004.</dd>
    <dt>Safari</dt>
    <dd>This browser was developed by Apple and is the default browser for iPhone, iPad and Mac devices.</dd>
    <dt>Brave</dt>
    <dd>This is a free web browser first released in 2016 that is based on the Chromium web browser.</dd>
  </dl>
  ```

---

## Step 7

The last browser you will add to the list will be for the `Arc` browser.

Add another `dt` element containing the text `Arc`.

Below your `dt` element, add a `dd` element with the following text: `This is a free Chromium based web browser first released in 2023 by The Browser Company.`

- **Codigo**

  ```html
  <h1>List of Major Web Browsers</h1>
  <dl>
    <dt>Google Chrome</dt>
    <dd>This is a free web browser developed by Google and first released in 2008.</dd>
    <dt>Firefox</dt>
    <dd>This is a free web browser developed by the Mozilla Corporation and first created in 2004.</dd>
    <dt>Safari</dt>
    <dd>This browser was developed by Apple and is the default browser for iPhone, iPad and Mac devices.</dd>
    <dt>Brave</dt>
    <dd>This is a free web browser first released in 2016 that is based on the Chromium web browser.</dd>
    <dt>Arc</dt>
    <dd>This is a free Chromium based web browser first released in 2023 by The Browser Company.</dd>
  </dl>
  ```

---

**With that last addition, your browser list is complete!**
