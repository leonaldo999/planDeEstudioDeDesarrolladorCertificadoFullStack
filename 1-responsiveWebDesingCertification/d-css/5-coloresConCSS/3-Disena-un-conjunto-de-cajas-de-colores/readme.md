# Diseñar un conjunto de cajas de colores

En este laboratorio, practicará el uso de colores CSS mediante el diseño de cuadros.

**Objetivo**: Cumplir las historias de usuario a continuación y lograr aprobar todas las pruebas para completar el laboratorio.

---

## Historias de usuarios

- Debes establecer el color de fondo de body en #f4f4f4.

- Debes tener una div con clase color-grid para contener todos tus elementos de color.

- Debes tener cinco elementos div dentro del elemento .color-grid.

- Cada uno de los cinco elementos div debe tener una clase de color-boxy color#, donde # es el número del orden de ese div. Por ejemplo: color1 para el primero div, color2 para el segundo , y así sucesivamente.

- La clase .color-box debe tener width y height para que sus elementos div sean visibles en la página.

- El elemento .color1 debe tener un valor background-color de color hexadecimal.

- El elemento .color2 debe tener un background-color que utilice un valor de color RGB.

- El elemento .color3 debe tener un background-color que utilice un valor de color predefinido (palabra).

- El elemento .color4 debe tener un background-color que utilice un valor de color HSL.

- El elemento .color5 debe tener un background-color set.

Nota: asegúrese de vincular su hoja de estilo en su HTML y aplicar su CSS.
