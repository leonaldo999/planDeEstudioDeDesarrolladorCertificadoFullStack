# Cree una página de folleto de eventos

> Objetivo: Cumpla con las historias de usuario que se indican a continuación y apruebe todas las pruebas para completar el laboratorio.

## Historias de usuario

- Debe tener un elemento de `header` dentro del cuerpo.

- El elemento de `header` debe contener una imagen del evento y un `h1` con el título del evento, en ese orden. Si lo desea, puede utilizar esta imagen: `https://cdn.freecodecamp.org/curriculum/labs/event.jpg`

- Debes tener un elemento `main` dentro del cuerpo.

- Tu elemento `main` debe tener al menos dos elementos de `section` dentro que muestren las características del evento.

- Cada uno de tus elementos de `section` debe tener un `h2` dentro.

- Tu `body` debe tener un relleno superior e inferior de `50px`.

- Tu `body` debe tener un margen superior e inferior de `0`, y un margen izquierdo y derecho que se centre por sí mismo.

- Tu `body` debe tener un ancho establecido en relación con el ancho de la ventana gráfica.

- El `body` debe utilizar la función `calc` para establecer su propiedad `min-height` en el 100 % de la altura de la ventana gráfica menos todo el relleno aplicado a la parte superior e inferior del `body`.

- Debe tener al menos un `hr` dentro de su folleto.

- Debe establecer el `width` de todos los elementos `hr` y `section` en un valor porcentual relativo a su elemento principal.

>[!note]
> Asegúrese de vincular su hoja de estilos en su HTML y aplicar su CSS.
