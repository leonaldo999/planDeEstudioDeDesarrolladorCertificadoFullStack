# Crea un portafolio personal

En este proyecto, crearás un portafolio personal utilizando HTML y CSS. El portafolio debe incluir una sección de introducción, una sección de proyectos y una sección de contacto. Además, debes agregar animaciones para hacer que el portafolio sea más atractivo visualmente.

---

## Historias de usuario

1. Tu portafolio debe tener una sección de bienvenida con el `id` de `welcome-section`.

2. La sección de bienvenida debe tener un elemento `h1` que contenga texto.

3. Tu portafolio debe tener una sección de proyectos con el `id` de `project-section`.

4. La sección de proyectos debe contener al menos un elemento con la clase `project-tile` para mostrar un proyecto.

5. La sección de proyectos debe contener al menos un enlace a un proyecto.

6. Tu portafolio debe tener una barra de navegación con el identificador `navbar`.

7. La barra de navegación debe contener al menos un enlace en el que se pueda hacer clic para navegar a diferentes secciones de la página.

8. Tu portafolio debe tener un enlace con un `id` de `profile-link`, que abra tu perfil de GitHub o freeCodeCamp en una nueva pestaña.

9. Tu portafolio debe tener al menos una consulta de `media query`.

10. La altura de la sección de bienvenida debe ser igual a la altura de la ventana de visualización.

11. La barra de navegación debe estar siempre en la parte superior de la ventana de visualización.

Nota: Asegúrate de vincular tu hoja de estilos en tu HTML y aplicar tu CSS.
