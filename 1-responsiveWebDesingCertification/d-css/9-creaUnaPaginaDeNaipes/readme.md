# Crea una página de naipes

---

## Historias de usuario

1. Debes crear una página web que muestre al menos tres cartas.

2. Debes tener un elemento `main` con un ID de `playing-cards`.

3. Dentro de tu elemento `#playing-cards`, debes tener al menos tres elementos `div`, cada uno con una clase de `card`.

4. Dentro de cada elemento `.card`, debes tener tres elementos `div`, el primero con una clase de `left`, el segundo con una clase de `middle` y el tercero con una clase de `right`.

5. Tu elemento `#playing-cards` debe usar flexbox para centrar horizontalmente sus elementos secundarios, permitir que se ajusten y poner un espacio de `20px` entre ellos.

6. Cada uno de tus elementos `.card` debe usar flexbox para justificar sus elementos secundarios usando `space-between`.

7. Cada uno de tus elementos `.left` debe usar las propiedades de los elementos flexbox para alinearse al principio de su contenedor principal.

8. Cada uno de tus elementos `.middle` debe usar las propiedades de los elementos flexbox para alinearse en el centro de su contenedor principal.

9. Cada uno de tus elementos `.right` debe usar las propiedades de los elementos flexbox para alinearse al final de su contenedor principal.

10. Cada uno de tus elementos `.middle` debe usar flexbox para mostrar sus elementos secundarios en una columna.

>[!note]
> Aquí tienes algunos caracteres que puedes copiar y pegar para usar en tus tarjetas si lo deseas: `♠`, `♣`, `♥` y `♦`.
