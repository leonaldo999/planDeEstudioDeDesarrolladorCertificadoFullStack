# Diseña una tarjeta de entrada de blog

Crea una aplicación con una funcionalidad similar a la de este proyecto de ejemplo. Intenta no copiar el proyecto de ejemplo, dale tu propio estilo personal.

En este laboratorio, practicarás cómo aplicar estilos a fondos y bordes mediante la creación de una tarjeta de entrada de blog.

Objetivo: Cumple con las historias de usuario que se indican a continuación y supera todas las pruebas para completar el laboratorio.

---

## Historias de usuario

- Debes tener un elemento div con una clase blog-post-card para contener todos los elementos de tu tarjeta.

- Dentro del elemento .blog-post-card, debes tener un elemento de imagen con un atributo alt y un texto válidos, y una clase post-img. Puedes utilizar `https://cdn.freecodecamp.org/curriculum/labs/cover-photo.jpg` para el atributo src de la imagen.

- Debes tener un div con una clase post-content dentro del elemento .blog-post-card con lo siguiente:
  - Un elemento h2 con una clase post-title y texto para el título de tu entrada de blog.
  - Un elemento p con una clase post-excerpt y texto para resumir tu entrada de blog.
  - Un elemento a con una clase read-more con el texto Leer más.

- Debes establecer el color de fondo del elemento body en un valor distinto al blanco.

- Debes aplicar los siguientes estilos al elemento .blog-post-card:
  
  - Un fondo blanco.
  - Esquinas redondeadas.
  - Un ancho de tu elección.
  - La alineación del texto que prefieras.

- El elemento .post-img debe diseñarse de manera que la imagen ocupe todo el ancho de la tarjeta y tenga un borde inferior.

- El elemento .post-content debe diseñarse de manera que haya un relleno dentro de la tarjeta.

- Los elementos .post-title y .post-excerpt deben tener un color de texto distinto al predeterminado y márgenes en todos los lados.

- El elemento .read-more debe diseñarse como un botón y tener:
  
  - Un color de texto distinto al predeterminado.
  - Un color de fondo.
  - Márgenes en todos los lados.
  - Relleno en todos los lados.
  - Esquinas redondeadas.
  - Una propiedad de visualización establecida en inline-block.
  - Un efecto al pasar el cursor que cambie su color de fondo.

*Nota: Asegúrate de vincular tu hoja de estilos en tu HTML y aplicar tu CSS.*
