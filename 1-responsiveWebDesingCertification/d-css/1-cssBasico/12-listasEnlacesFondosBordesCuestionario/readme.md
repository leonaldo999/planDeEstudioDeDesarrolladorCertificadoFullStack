# Cuestionario Listas, enlaces, fondos y bordes

Para aprobar el cuestionario, debe responder correctamente al menos 18 de las 20 preguntas siguientes.

---

- **1. ¿Qué tipo de valores puede aceptar la propiedad `line-height`?**
  - [ ] a) Solo valores de palabras clave como `inherit`
  - [ ] b) Solo valores de píxeles
  - [ ] c) Solo unidades em
  - [x] d) Números, porcentajes y unidades de longitud // ❎

- **2. ¿Cuál de los siguientes no es un valor válido para la `line-height`?**
  - [ ] a) `2em`
  - [ ] b) `150%`
  - [x] c) `bold` // ❎
  - [ ] d) `normal`

- **3. ¿Qué controla el `list-style-type`?**
  - [ ] a) El espaciado de los elementos de la lista
  - [ ] b) La posición de un marcador de lista
  - [ ] c) La alineación de los elementos de la lista
  - [x] d) El símbolo marcador utilizado en una lista // ❎

- **4. ¿Qué valor no es válido para `list-style-type`?**
  - [ ] a) `square`
  - [ ] b) `circle`
  - [ ] c) `disc`
  - [x] d) `hexagon` // ❎

- **5. ¿Cuáles son los dos valores aceptables para `list-style-position`?**
  - [ ] a) `inline` y `block`
  - [ ] b) `top` y `bottom`
  - [ ] c) `left` y `right`
  - [x] d) `inside` y `outside` // ❎

- **6. ¿Qué propiedad CSS añade espacio fuera de cada elemento de la lista para mejorar la legibilidad?**
  - [ ] a) `padding`
  - [x] b) `margin` // ❎
  - [ ] c) `line-height`
  - [ ] d) `text-indent`

- **7. ¿A qué apunta la pseudoclase `:link`?**
  - [x] a) Enlaces no visitados // ❎
  - [ ] b) Todos los hipervínculos
  - [ ] c) Enlaces flotantes
  - [ ] d) Enlaces ya visitados

- **8. ¿Qué pseudo-clase aplica estilos a un enlace cuando el usuario ya ha hecho clic en él?**
  - [x] a) `:visited` // ❎
  - [ ] b) `:focus`
  - [ ] c) `:hover`
  - [ ] d) `:link`

- **9. ¿Qué pseudo-clase aplica estilos a un elemento cuando obtiene el foco de entrada?**
  - [ ] a) `:visited`
  - [x] b) `:focus` // ❎
  - [ ] c) `:hover`
  - [ ] d) `:link`

- **10. ¿Cuándo se aplica la pseudo-clase `:active`?**
  - [ ] a) Cuando el navegador termina de renderizar
  - [ ] b) Cuando el usuario no está interactuando
  - [x] c) Cuando un usuario hace clic o toca el elemento // ❎
  - [ ] d) Cuando se carga el enlace por primera vez

- **11. ¿Qué pseudoclase se utiliza para los efectos al pasar el cursor?**
  - [ ] a) `:link`
  - [x] b) `:hover` // ❎
  - [ ] c) `:click`
  - [ ] d) `:active`

- **12. ¿Qué hace `background-size: cover`?**
  - [ ] a) Fija la imagen en una posición.
  - [ ] b) Recorta la imagen.
  - [ ] c) Repite la imagen en todas las direcciones.
  - [x] d) Escala la imagen para llenar el elemento conservando la relación de aspecto. // ❎

- **13. ¿Qué valor de `background-repeat` repetirá la imagen de fondo tanto horizontal como verticalmente?**
  - [ ] a) `repeat-x`
  - [ ] b) `no-repeat`
  - [x] c) `repeat` // ❎
  - [ ] d) `space`

- **14. ¿Qué propiedad establece dónde se coloca la imagen de fondo?**
  - [x] a) `background-position` // ❎
  - [ ] b) `background-location`
  - [ ] c) `background-place`
  - [ ] d) `background-origin`

- **15. ¿Qué valor de `background-attachment` mantiene la imagen fija al desplazarse?**
  - [ ] a) `static`
  - [ ] b) `scroll`
  - [x] c) `fixed` // ❎
  - [ ] d) `sticky`

- **16. ¿Cuál de las siguientes opciones establece correctamente una imagen de fondo que no se repite y está centrada?**
  - [x] a) `background: center no-repeat url("img.png");` // ❎
  - [ ] b) `background: image url("img.png") center;`
  - [ ] c) `background: no-repeat-center img("img.png");`
  - [ ] d) `background-position: center no-repeat url("img.png");`

- **17. ¿Qué propiedad se utiliza para establecer los bordes superior, derecho, inferior e izquierdo en una sola declaración?**
  - [x] a) `border` // ❎
  - [ ] b) `outline`
  - [ ] c) `border-radius`
  - [ ] d) `border-style`

- **18. ¿Qué efecto tiene establecer `border-radius: 10px;` en un elemento?**

  - [ ] b) Crea un borde de 10-píxeles.

- **19. ¿Cuál de las siguientes reglas CSS crea correctamente un degradado lineal de rojo a amarillo?**

  - [ ] b) `background: gradient-linear(red to yellow);`
  - [ ] c) `background-color: gradient(red, yellow);`

- **20. ¿Cuál de las siguientes declaraciones CSS aplica correctamente un degradado radial desde el blanco en el centro hasta el azul en los bordes?**

  - [ ] b) `background: radial-gradient(blue, white);`
