# Diseña una tarjeta de presentación

Cree una aplicación que sea funcionalmente similar a este proyecto de ejemplo. Intente no copiar el proyecto de ejemplo, dele su propio estilo personal.

Objetivo: Cumpla con las historias de usuario que se indican a continuación y apruebe todas las pruebas para completar el laboratorio.

---

## Historias de usuario

1- Debes tener un div con un atributo de clase con un valor de business-card que contenga todos los demás elementos.

2- Dentro del elemento .business-card, debe haber un elemento img con un atributo de clase con un valor de profile-image. Si lo deseas, puedes establecer la fuente de la imagen en `https://cdn.freecodecamp.org/curriculum/labs/flower.jpg`. Debe haber un alt con una descripción significativa.

3- Dentro del elemento .business-card, después del elemento img, debe tener tres elementos p con un atributo de clase que tenga, respectivamente, un valor de full-name, designation y company.

4- El primer elemento p debe contener su nombre.

5- El segundo elemento p debe contener su cargo.

6- El tercer elemento p debe contener el nombre de su empresa.

7- Debe haber un elemento hr debajo del tercer elemento p.

8- Después del elemento hr, debe haber dos elementos p. En el primer elemento p, debe tener una dirección de correo electrónico como texto. En el segundo elemento p, el texto debe ser un número de teléfono.

9- Después de los dos elementos p, debe haber un elemento a con una clase de portfolio-link, el texto Portfolio, y debe enlazar con una URL válida.

10- Debe haber un elemento hr después del elemento a que contiene el enlace del portfolio.

11- Debes tener otro elemento div con un atributo de clase con el valor social-media. Dentro de este elemento, debe haber un elemento h2 con el texto Conéctate conmigo.

12- Dentro de los elementos .social-media, debe haber tres elementos a. Los elementos a deben tener el texto Twitter, LinkedIn y GitHub respectivamente con enlaces a sitios web válidos.

13- Debes enlazar la hoja de estilos styles.css utilizando la etiqueta link en la sección head del archivo HTML.

14- El fondo de tu página debe ser rosybrown. La fuente general debe ser Arial, con sans-serif como alternativa.

15- El selector .business-card debe tener propiedades para establecer el elemento con un ancho de 300 píxeles y un color de fondo de tu elección. También debes establecer un relleno de 20 píxeles alrededor y un margen superior de 100 píxeles. El texto debe estar alineado al centro y el tamaño de la fuente debe ser de 16 píxeles. Los márgenes izquierdo y derecho deben establecerse en automático.

16- El selector .profile-image debe tener una propiedad max-width con un valor del 100 %.

17- Todos los elementos p deben tener un margen superior e inferior de 5 píxeles.

18- Ninguno de los enlaces de la página debe estar subrayado.
