🚀 Introducción

Revisión de los selectores de atributos CSS para seleccionar elementos HTML en función de sus atributos, como href o title. Aprenderemos a usar diferentes tipos de selectores de atributos para aplicar estilos específicos a elementos que cumplen ciertas condiciones.

Trabajar con diferentes selectores de atributos y enlaces

  • Definición: El selector de atributos le permite seleccionar elementos HTML en función de sus atributos, como los atributos href o title. 
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org">Link with href</a>
<a>No href</a>
a[href] {
  color: blue;
  text-decoration: underline;
}

a {
  display: block;
  margin-bottom: 10px;
  font-family: sans-serif;
  color: gray;
}
Link with href (Active) No href (Inactive)

💡 Análisis: El primer enlace es azul y subrayado porque el selector a[href] detectó la presencia del atributo. El segundo se mantiene gris.

  • Atributo title: este atributo proporciona información adicional sobre un elemento. A continuación se muestra cómo se pueden seleccionar enlaces con el atributo title: 
<link rel="stylesheet" href="styles.css">

<a href="#" title="Tooltip">Link with title</a>
<a href="#">Normal link</a>
a[title] {
  font-weight: bold;
  text-decoration: none;
  color: #e67e22;
}

a {
  display: block;
  margin-bottom: 10px;
  font-family: sans-serif;
}
Link with title (Hover me!) Normal link

💡 Revisión: Solo el enlace con el atributo title recibe el peso de fuente negrita y el color especial.

  • Combina selectores para encontrar enlaces que tengan los atributos href y title: Combina varios selectores de atributos. 
<link rel="stylesheet" href="styles.css">

<a href="#" title="Info">Href + Title</a>
<a href="#">Only href</a>
a[href][title] {
  color: green;
  font-weight: bold;
}

a {
  display: block;
  margin-bottom: 10px;
  font-family: sans-serif;
  color: black;
}
Href + Title (Estilizado) Only href (Normal)

💡 Revisión Full Stack: El encadenamiento de atributos aumenta la precisión sin necesidad de crear clases CSS adicionales en tu HTML.

  • Busca una sola palabra dentro de una lista separada por espacios: busca enlaces que tengan una palabra de clase específica. 
<link rel="stylesheet" href="styles.css">

<a class="link primary">Primary link</a>
<a class="link secondary">Secondary link</a>
a[class~="primary"] {
  color: red;
  font-weight: bold;
}

a {
  display: block;
  margin-bottom: 10px;
  font-family: sans-serif;
  text-decoration: none;
  color: #333;
}
Primary link (Match!) Secondary link (No match)

💡 Dato de revisión: El selector [class~="primary"] es más flexible que [class="primary"], ya que este último fallaría si hay más de una clase presente.

  • Buscar valores que empiecen por un prefijo específico: Busca enlaces que empiecen por https://. 
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org">HTTPS link</a>
<a href="http://www.freecodecamp.org">HTTP link</a>
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}

a {
  display: block;
  margin-bottom: 12px;
  font-family: sans-serif;
  color: #e74c3c; /* Rojo para los no seguros en esta prueba */
}
HTTPS link (Secure) HTTP link (Not secure)

💡 Revisión: Observa cómo el selector ignoró el enlace http. Solo el que comienza exactamente con https:// activó la regla verde y el candado.

  • Buscar valores que terminen con un sufijo específico: Busca enlaces que terminen en .jpg. 
<link rel="stylesheet" href="styles.css">

<a href="photo.jpg">Image link</a>
<a href="index.html">HTML link</a>
a[href$=".jpg"] {
  color: darkgreen;
  text-decoration: underline dotted;
}

a {
  display: block;
  margin-bottom: 12px;
  font-family: sans-serif;
  color: #333;
}
Image link (.jpg) HTML link (.html)

💡 Revisión: Solo el enlace que termina exactamente en .jpg recibió el color verde oscuro y el subrayado punteado.

  • Coincidir valores que contengan una subcadena en cualquier lugar: busca enlaces que contengan https en cualquier parte del valor. 
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org">Secure link</a>
<a href="page.html?ref=https">Local link with https param</a>
<a href="page.html">Normal Local link</a>
a[href*="https"] {
  color: teal;
  font-weight: bold;
}

a {
  display: block;
  margin-bottom: 12px;
  font-family: sans-serif;
  text-decoration: none;
  color: #333;
}
Secure link (Matches at start) Local link with https param (Matches in middle) Normal Local link (No match)

💡 Análisis de Revisión: Fíjate cómo el segundo enlace se pintó de color teal aunque no empieza con "https". El asterisco lo encontró perdido dentro de la URL.

  • Resumen: Estos patrones facilitan el diseño coherente de los enlaces en función de sus atributos y valores.

Selección de elementos con los atributos lang y data-lang

  • Atributo lang: Este atributo se utiliza en HTML para especificar el idioma del contenido de un elemento. Es posible que desee aplicar estilos diferentes a los elementos en función del idioma en el que estén escritos, especialmente en un sitio web multilingüe. 
<link rel="stylesheet" href="styles.css">

<p lang="en">English text</p>
<p lang="fr">French text</p>
p[lang="en"] {
  font-style: italic;
  color: #2c3e50;
}

p {
  font-family: sans-serif;
  margin-bottom: 10px;
}

English text (Italicized by CSS)

French text (Standard style)

💡 Revisión: El selector detecta el código de idioma y aplica el estilo automáticamente. Esto es mejor que usar clases como .english-text porque ya estás usando un atributo estándar de HTML.

  • Atributo data-lang: Los atributos de datos personalizados, como el atributo data-lang, se utilizan habitualmente para almacenar información adicional en los elementos, como especificar el idioma utilizado en una sección concreta del texto. A continuación se muestra cómo se puede aplicar estilo a los elementos basándose en el atributo data-lang: 
<link rel="stylesheet" href="styles.css">

<div data-lang="fr">French content</div>
<div data-lang="en">English content</div>
div[data-lang="fr"] {
  color: blue;
  font-weight: bold;
}

div {
  padding: 10px;
  margin-bottom: 5px;
  font-family: sans-serif;
}
French content (Applied style)
English content (Standard style)

💡 Dato de revisión: Al usar data-*, el navegador no aplica comportamientos por defecto (como lo haría con lang), dándote control total sobre el diseño.

Trabajar con el selector de atributos, elementos de listas ordenadas y el atributo type

  • Atributo type: al trabajar con listas ordenadas en HTML, el atributo type permite especificar el estilo de numeración utilizado, como numérico, alfabético o números romanos. 
<link rel="stylesheet" href="styles.css">

<ol type="A">
  <li>Alpha</li>
  <li>Beta</li>
</ol>

<ol type="i">
  <li>One</li>
  <li>Two</li>
</ol>
ol[type="A"] {
  color: purple;
  font-weight: bold;
}

ol[type="i"] {
  color: darkgreen;
  font-style: italic;
}
  1. Alpha
  2. Beta
  1. One
  2. Two

💡 Análisis: El CSS seleccionó automáticamente cada lista por su atributo type y le aplicó un estilo distinto.
