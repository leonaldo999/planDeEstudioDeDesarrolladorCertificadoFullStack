# Crear una página de correo electrónico confidencial

---

## Historias de usuario

1. Debes tener un elemento `main` con un `id` de `email`.

2. Tu elemento `#email` debe tener un `padding` de `50px`, un margen superior de `50px`, un ancho de `500px` y un borde de `2px`.

3. El ancho total de tu elemento `#email`, incluyendo rellenos y bordes, debe ser de `500px`.

4. Debes tener dos elementos `div`, uno con un id de `confidential` y otro con un id de `top-secret`, dentro de tu elemento `#email`.

5. Tus elementos `#confidential` y `#top-secret` deben tener una `display` en `inline-block`.

6. Tus elementos `#confidential` y `#top-secret` deben tener un `padding`, un `margin-left` y un `border`.
7. El elemento `#confidential` debe tener el texto `CONFIDENTIAL`.

8. El elemento `#top-secret` debe tener el texto `TOP SECRET`.

9. Tus elementos `#confidential` y `#top-secret` deben girarse utilizando una transformación CSS.

10. Debes tener al menos tres elementos de párrafo dentro de tu elemento `#email`.

11. Debes tener al menos tres elementos `span` con una clase de `blurred`, dentro de tus elementos de párrafo.

12. Debes tener un selector `blurred` que difumine el elemento `3px` utilizando un filtro CSS.
