# Diseñar un formulario de contacto

---

## Historias de usuario

1. Debes tener un `div` con una clase de `form-container` para alojar tu formulario.

2. Debes tener un elemento de `form` dentro del elemento .`form-container`.

3. Dentro del elemento de `form`, debes tener los siguientes elementos y campos de entrada:

    - Un elemento `h2` con algo de texto.
    - Un campo de text input para el nombre con el `type` establecido en `text` y los atributos `id`, `name` y `required`.
    - Un email input con el `type` establecido en `email` y los atributos `id`, `name` y `required`.
    - Un `textarea` para el mensaje con los atributos `id`, `name` y `required`.
    - Un elemento `button` con el texto `"Submit"` y un atributo `type` de `submit`.

4. Cada elemento `input` y `textarea` debe tener su elemento `label` correspondiente con un atributo `for` establecido en el `id` del elemento.

5. El elemento `.form-container` debe tener un `background-color` y valores para `border-radius`, `padding` y `width`.

6. Los elementos `label` deben tener un margen y un color de fuente.

7. Los elementos `input` y `textarea` deben tener valores para:

    - `width`.
    - `padding`.
    - `Margin` en la parte inferior.

8. El elemento `button` debe tener:

    - Color de fondo.
    - Tamaño de fuente distinto al predeterminado.

9. El botón debe tener un efecto hover que cambie el color de fondo.

>[!note]
>Asegúrate de vincular tu hoja de estilos en tu HTML y aplicar tu CSS.
