# Diseña una página de selección de funciones

**Objetivo**: Cumplir las historias de usuario a continuación y lograr aprobar todas las pruebas para completar el laboratorio.

---

## Historias de usuario

1. Tu página debe tener un elemento `h1` con el texto `Feature Selection`.

2. Tu página debe tener un elemento `div` con la clase `feature-card-container`.

3. Tu página debe tener al menos dos elementos `label`, cada uno con la clase `feature-card` dentro del `div`.

4. Cada uno de tus elementos `label` debe tener un texto de etiqueta y un elemento `input` de tipo `checkbox`.

5. Cree un selector que apunte a las casillas de verificación y aplique el siguiente estilo:
   - Todos sus elementos de casilla de verificación deben estar configurados con `appearance: none;` en su CSS.
   - Todos sus elementos de casilla de verificación deben tener un ancho de borde, color y estilo de su elección.

6. Cuando la casilla de verificación esté marcada, debe mostrar una marca de verificación `✓` en el centro de la casilla.

7. Cuando la casilla de verificación esté marcada, el color de fondo de la casilla debe cambiar al color que elijas.

8.Cuando la casilla de verificación esté marcada, el color del borde de la casilla debe cambiar al color que elijas.

> [!NOTE]
> Asegúrate de vincular tu archivo CSS en tu HTML.
