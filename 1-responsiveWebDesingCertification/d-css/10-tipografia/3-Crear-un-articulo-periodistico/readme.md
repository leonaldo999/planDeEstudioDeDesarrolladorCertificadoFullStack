# Crear un artículo periodístico

---

Historias de usuario:

1. Debes establecer el `font-size` del `main` de tu documento HTML en `24px`.

2. Debes tener un elemento con una clase de `newspaper` que contenga todos los demás elementos.

3. Tu elemento `.newspaper` debe tener un `font-size` de `16px` y una fuente `Open Sans` con una fuente alternativa `sans-serif`.

4. Dentro de tu elemento `.newspaper`, debes tener al menos siete elementos más: uno para el nombre del periódico que tenga una clase de `name`, uno para la fecha del artículo con una clase de `date`, uno para el titular con una clase de `headline`, uno para el subtítulo con una clase de `sub-headline`, uno para el autor con una clase de `author` y dos párrafos, cada uno con una clase de `text`. Todos estos elementos deben rellenarse con la información de tu artículo.

5. Tu elemento `.name` debe tener un `font-size` que sea el doble del `font-size` del elemento `root` y debe utilizar la fuente `Times New Roman`, con una fuente alternativa `serif`.

6. Tus elementos `.name` y `.author` deben utilizar CSS para que todos sus caracteres estén en mayúsculas.

7. El elemento `.headline` debe tener un `font-size` que sea el doble del `font-size` de su elemento padre y debe estar en `bold`.

8. El elemento `.sub-headline` debe tener un `font-weight` de `100`, un `font-size` que sea `1,5` veces el `font-size` de su elemento padre y debe estar en cursiva.

9. Tu `.author` debe usar CSS para ponerlo en `bold`.

10. Tus elementos `.text` deben tener una sangría de texto de `20px`.

11. Tus elementos `.text` deben tener una `line-height` que sea el doble del `font-size` de su elemento padre.

12. La primera letra de tus elementos `.text` debe estar en `bold` y tener el doble del `font-size` de su elemento padre. Usa el selector `::first-letter` para ello.
