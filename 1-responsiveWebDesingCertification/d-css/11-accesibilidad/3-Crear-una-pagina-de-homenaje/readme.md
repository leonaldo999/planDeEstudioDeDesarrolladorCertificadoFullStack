# Crear una página de homenaje

---

## Historias de usuarios

1. Tu página de homenaje debe tener un elemento `main` con el `id` correspondiente `main`, que contenga todos los demás elementos.

2. Debes ver un elemento con el `id` `title`, que contiene una cadena (es decir, texto) que describe el tema de la página de homenaje (por ejemplo, "Dr. Norman Borlau").

3. Debes ver un elemento `figure` o `div` con el `id` `img-div`.

4. Dentro del elemento `#img-div`, deberías ver un elemento `img` con un `id="image"` correspondiente.

5. Dentro del elemento `#img-div`, deberías ver un elemento con un `id="img-caption"` correspondiente que contiene contenido textual que describe la imagen que se muestra en `#img-div`.

6. Deberías ver un elemento con un `id="tribute-info"` correspondiente, que contiene contenido textual que describe el tema de la página de homenaje.

7. Deberías ver un elemento `a` con un `id="tribute-link"` correspondiente, que enlaza con un sitio externo que contiene información adicional sobre el tema de la página de homenaje. SUGERENCIA: Debes asignar a tu elemento un atributo de `target` y establecerlo en `_blank` para que tu enlace se abra en una nueva pestaña.

8. Tu `#image` debe utilizar las propiedades `max-width` y `height` para cambiar de tamaño de forma responsiva, en relación con el ancho de su elemento padre, sin exceder su tamaño original.

9. Tu elemento `img` debe estar centrado dentro de su elemento padre.

[!NOTE]
> Asegúrate de enlazar tu hoja de estilos en tu HTML y aplicar tu CSS.
